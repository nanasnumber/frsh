# FRSH (FRSH Required! Serve Hot!)
FRSH is a minimal livereload sever.

## How to use it
- `cd path/to/project`
- Execute command `frsh` within your project folder
- Live reload server would be serve at `localhost:8080`

## Requirement
```
go 1.16
```

## Build from source
```
make build
```

## Installation FRSH
```
make install
```

## Uninstall FRSH
```
make uninstall
```
